# Code of Conduct

This project follows a simple contributor covenant:

- Be respectful and constructive.
- Focus criticism on code, design, documentation, and behavior, not people.
- Do not publish private subscription data, credentials, or personal infrastructure details.
- Assume good intent, but accept corrections quickly.
- Maintainers may remove disruptive, abusive, or unsafe content from project spaces.

If the project adopts a formal community governance model later, this file can be replaced with a fuller code of conduct.
